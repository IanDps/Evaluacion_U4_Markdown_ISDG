Preguntas a responder:
1- ¿Qué es un archivo Markdown y para que sirve?
Es un lenguaje de marcado ligero que se utiliza para darle estilos al texto para darle el formato de negrita cursiva esto gracias a su creador en 2004 por John Gruber utilizando caracteres sencillo en un plano. Por lo general sirve para crear una documentación adecuada y legible para el programador y los navegadores web, siendo que se utiliza como estándar en varias plataformas como el propio GitHub.

2- ¿Cuál es el propósito del archivo README.md en un repositorio?
_El archivo README.md dentro del repositorio se utiliza para ser como la presentación de un proyecto permitiendo dar una descripción detallada, de las instrucciones de la instalación, el uso de esta y cualquier información extra que sea considerada relevante para que las personas que entren al repositorio sepan el propósito de este mismo._
